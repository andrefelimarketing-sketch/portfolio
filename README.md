# Retro Portfolio (HTML/CSS/JS)

## Run locally
Open `index.html` in your browser, or use a local server:

- VS Code extension: "Live Server"
- Or Python: `python -m http.server 5500`

## Replace placeholders
- Web Dev tab: replace `https://example.com` with the real Plus500 project URL.
- Add more images to `/assets` and duplicate a `<figure class="shot">...</figure>` block.

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. Repo Settings → Pages → Deploy from branch → `/ (root)` → Save.
